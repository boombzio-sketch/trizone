📖 TRIZONE 소개

TRIZONE은 수영·사이클·런 세 종목을 아우르는 트라이애슬론 선수와 동호인을 위한 훈련 기록 및 소셜 커뮤니티 플랫폼입니다. 개인 훈련 기록 관리부터 클럽 활동, 대회 일정 확인, 관리자 문의까지 트라이애슬론 라이프사이클 전체를 하나의 앱에서 경험할 수 있습니다.

🚀 시작하기

회원가입

  1. 앱 하단의 로그인 화면에서 회원가입 탭을 선택합니다.
  2. 닉네임, 이메일, 비밀번호를 입력합니다.
  3. 가입하기 버튼을 누르면 즉시 계정이 생성되고 로그인됩니다.
Tip 닉네임은 피드와 랭킹에서 다른 회원에게 공개됩니다. 실명 또는 운동 닉네임을 사용하세요.

로그인

  1. 이메일과 비밀번호를 입력합니다.
  2. 로그인 버튼을 누르면 30일간 로그인 상태가 유지됩니다.

비밀번호 찾기

이메일이 등록된 계정은 관리자 없이 직접 재설정할 수 있습니다.

  1. 로그인 화면 하단의 비밀번호를 잊으셨나요? 링크를 탭합니다.
  2. 가입 시 사용한 이메일(또는 닉네임)을 입력하고 코드 전송을 누릅니다.
  3. 이메일로 받은 6자리 코드와 새 비밀번호를 입력합니다.
  4. 비밀번호 재설정을 누릅니다.
주의 재설정 코드는 발송 후 30분간만 유효하며 한 번 사용하면 즉시 만료됩니다.
이메일 미등록 계정 이메일이 등록되지 않은 경우 관리자에게 직접 코드 발급을 요청해야 합니다. 마이페이지 → 관리자 문의를 이용하거나 관리자에게 연락하세요.

아바타 설정

가입 후 마이페이지 → 수정에서 아바타 색상(12가지)을 선택하거나 직접 업로드한 사진으로 프로필 이미지를 꾸밀 수 있습니다.

  • ⭐ 좋아요 누르기 / 취소

사용자 검색 & 팔로우

  1. 피드 상단 검색 아이콘을 탭합니다.
  2. 닉네임을 입력하면 실시간으로 결과가 표시됩니다.
  3. 프로필 카드에서 팔로우 버튼을 누릅니다.
참고 팔로우는 상대방의 승인 없이 즉시 적용됩니다. 언제든지 취소할 수 있습니다.

목록 탭 (📋 목록)

내가 등록한 훈련 기록을 날짜순으로 확인합니다. 각 카드에서 종목, 거리, 시간, 메모와 승인 상태를 확인할 수 있습니다.

  • 기록 카드 우측 🗑️ 버튼으로 기록 삭제 가능
  • 승인 상태 표시: 승인대기 / 승인 / 반려
  • 브릭 기록은 세그먼트별 거리가 요약 표시됨

달력 탭 (📅 달력) NEW

월별 달력 형태로 내 훈련 기록을 시각적으로 확인합니다.

  • 훈련을 기록한 날짜에 종목 아이콘(🏊🚴🏃⚡)이 표시됩니다
  • 오늘 날짜는 배경색으로 강조됩니다
  • 날짜를 탭하면 해당 날의 훈련 목록이 하단에 표시됩니다
  • 좌우 화살표(‹ ›)로 월 이동이 가능합니다
Tip 달력 탭에서 날짜를 다시 탭하면 선택이 해제됩니다.

클럽 직접 만들기 NEW

클럽장 신청이 승인되었거나 관리자인 경우, 클럽 화면 우측 상단의 + 클럽 만들기 버튼이 나타납니다. (이미 운영 중인 클럽이 있으면 비활성화됩니다.)

  1. + 클럽 만들기 버튼을 누릅니다.
  2. 클럽명을 입력하고, 지역을 선택합니다.
  3. 클럽 소개를 입력합니다 (선택사항).
  4. 🏊 클럽 만들기를 눌러 생성합니다.

클럽 가입 신청

  1. 전체 클럽에서 가입하고 싶은 클럽을 선택합니다.
  2. 🙋 클럽 가입 신청 버튼을 누릅니다.
  3. 클럽장이 승인하면 정식 회원이 됩니다.
참고 승인 전에는 클럽 상세 정보 카드에 ⏳ 클럽장 승인 대기 중 상태가 표시됩니다.

훈련 탭 — 달력 뷰 NEW

훈련 탭은 월별 달력 형태로 표시됩니다. 훈련이 등록된 날짜에는 작은 점이 표시됩니다.

  1. 달력에서 날짜를 탭하면 해당 날의 훈련 목록이 하단에 표시됩니다.
  2. 훈련 카드에서 상세 정보(시간, 장소, 설명, 참가 인원)를 확인합니다.
  3. 참가 신청이 가능한 훈련은 참가 신청 버튼이 활성화됩니다.
  4. 👥 N명 버튼을 탭하면 참가자 명단(참석/불참/신청 상태 포함)을 볼 수 있습니다.
  5. 훈련에 외부 링크가 있으면 🔗 상세보기 버튼이 표시됩니다.
주의 정원이 마감된 훈련(마감 표시)은 추가 신청이 불가능합니다.

클럽 탈퇴

클럽 상세 페이지 정보 카드의 탈퇴 버튼을 누릅니다. 탈퇴 후에는 클럽 전용 피드와 훈련에 접근할 수 없습니다.

프로필 편집

📋 공지사항

관리자가 등록한 공지·매뉴얼·안내문을 확인하는 게시판입니다. 마이페이지 → 📋 공지사항 버튼으로 진입합니다.

목록 보기

  • 📌 고정된 공지는 항상 최상단에 표시됩니다.
  • 제목, 작성자, 날짜가 표시되며 탭하면 상세 내용을 볼 수 있습니다.

상세 보기

  • 본문 텍스트와 첨부 사진을 확인합니다.
  • 사진을 탭하면 전체화면 뷰어가 열립니다.

사진 뷰어 조작

동작효과
핀치 (두 손가락)확대 / 축소

드래그확대된 이미지 이동
마우스 휠 (PC)확대 / 축소
이전 / 다음 버튼여러 장일 때 사진 전환

관리자 — 공지 작성

  1. 공지사항 화면 우측 상단 + 작성을 누릅니다.
  2. 제목과 내용을 입력합니다.
  3. 📷 사진 추가로 이미지를 첨부합니다 (최대 10장, 자동 압축).
  4. 📌 상단 고정을 켜면 목록 최상단에 고정됩니다.
  5. 💾 저장을 누릅니다.

🛡️ 클럽장 가이드

클럽장 또는 부클럽장 역할을 가진 회원을 위한 안내입니다.

클럽장 신청 안내 클럽장이 되려면 관리자에게 클럽장 신청을 해야 합니다. 관리자 승인 후 클럽 만들기 버튼이 활성화됩니다.

클럽 만들기

  1. 클럽 화면 우측 상단의 + 클럽 만들기를 탭합니다.
  2. 클럽명, 지역, 클럽 소개를 입력합니다.
  3. 🏊 클럽 만들기를 눌러 생성합니다.

회원 가입 승인

  1. 클럽 상세 → 관리 탭을 엽니다.
  2. 가입 대기 중인 회원 목록과 신청 메시지가 표시됩니다.
  3. 각 신청에 대해 ✓ 승인 또는 ✕ 거절을 선택합니다.

클럽 훈련 등록

  1. 클럽 상세 → 훈련 탭 하단의 + 훈련 등록을 누릅니다.
  2. 제목, 날짜(YYYY-MM-DD), 시간(선택), 장소, 설명, 모집 인원을 입력합니다.
  3. 등록을 누릅니다. 달력에 해당 날짜에 점이 표시됩니다.

참가자 출석 확인

훈련 카드의 👥 N명 버튼을 탭하면 참가자 명단을 볼 수 있습니다. 각 회원의 상태(신청/참석/불참)가 표시됩니다.

공지사항 작성

  1. 클럽 상세 → 공지사항 탭 → + 공지 작성을 누릅니다.
  2. 제목과 내용을 입력하고 게시를 누릅니다.

부클럽장 임명 / 해제 NEW

  1. 클럽 상세 → 회원 탭에서 대상 회원을 찾습니다.
  2. 회원 행 우측의 부클럽장 버튼을 누릅니다. 이미 부클럽장이면 해제 버튼이 표시됩니다.

부클럽장은 훈련 등록·관리, 가입 신청 승인·거절 권한을 가집니다.

회원 강제 탈퇴 NEW

  1. 클럽 상세 → 회원 탭에서 대상 회원을 찾습니다.
  2. 회원 행 우측의 탈퇴 버튼(빨간색)을 누르고 확인합니다.

클럽장 양도

  1. 클럽 상세 → 관리 탭 하단의 👑 클럽장 양도 → 양도하기를 누릅니다.
  2. 양도받을 회원을 목록에서 선택합니다.
  3. 👑 클럽장 양도 확정을 누릅니다. 양도 후 본인은 일반 회원으로 전환됩니다.

클럽 정보 수정 / 삭제

클럽 상세 상단 카드의 ✏️ 수정 버튼을 누르면 클럽명, 지역, 소개를 수정하거나 클럽 전체를 삭제할 수 있습니다.

주의 클럽 삭제 시 클럽 관련 모든 데이터(공지, 훈련, 회원 정보)가 영구 삭제됩니다.

⚙️ 관리자 가이드

관리자 역할 또는 승인관리자 권한을 가진 회원을 위한 안내입니다. 관리 화면은 하단 탭의 ⚙️ 관리에서 접근합니다.

탭 구성

탭접근 권한내용
훈련 승인관리자 + 승인관리자대기 중인 훈련 기록 승인/반려
클럽장 신청관리자 전용클럽장 신청 승인/거절
회원 관리관리자 전용전체 회원 조회 및 역할 변경

훈련 승인 (승인관리자 포함)

  1. 관리 → 훈련 승인 탭을 엽니다.
  2. 대기 중인 훈련 목록을 확인합니다.
  3. 각 기록의 종목, 거리, 시간, 메모를 검토합니다.
  4. ✓ 승인 또는 ✕ 반려를 선택합니다.

가입 신청 관리

  1. 관리 → 가입 신청 탭을 엽니다.
  2. 가입 신청 목록에서 신청자 정보와 메시지를 확인합니다.
  3. ✓ 승인 또는 ✕ 거절을 처리합니다.
참고 클럽장이 있는 클럽은 클럽장이 1차로 처리합니다. 클럽장이 없는 경우 관리자가 처리합니다.

회원 관리

  1. 관리 → 회원 관리 탭을 엽니다.
  2. 전체 회원 목록과 각 회원의 이메일, 훈련 횟수를 확인합니다.

회원 관리에서 할 수 있는 작업:

기능설명
관리자 권한 부여일반 회원 → 관리자로 역할 변경
관리자 권한 해제관리자 → 일반 회원으로 역할 변경
회원 삭제해당 회원의 모든 데이터 영구 삭제 (복구 불가)
주의 회원 삭제는 훈련 기록, 댓글, 팔로우 관계 등 모든 관련 데이터가 영구 삭제됩니다. 신중하게 진행하세요.

비밀번호 재설정 코드 발급 (이메일 미등록 회원)

이메일이 등록된 회원은 로그인 화면에서 직접 재설정할 수 있습니다. 이메일이 없는 회원은 관리자가 코드를 발급해 전달해야 합니다.

  1. 관리 → 회원 관리 탭을 엽니다.
  2. 해당 회원 행의 코드발급 버튼(노란색)을 탭합니다.
  3. 이메일이 등록된 회원은 자동으로 이메일 발송되며, 미등록 회원은 코드가 화면에 표시됩니다.
  4. 코드를 사용자에게 직접 전달합니다. 코드는 30분간 유효합니다.
Tip 코드는 한 번만 사용할 수 있습니다. 사용자가 재설정을 완료하면 자동으로 만료됩니다.

대회 등록 및 수정

관리자는 대회 캘린더 화면에서 직접 대회를 등록·수정·삭제할 수 있습니다.

  1. 대회 화면 우측 상단의 + 등록 버튼을 누릅니다.
  2. 대회명, 날짜, 장소, 종목(Sprint/Olympic/Half/King)을 입력합니다.
  3. 접수 시작일·마감일, 참가비, 모집 인원, 접수 URL을 입력합니다.
  4. 저장을 눌러 등록합니다.

❓ 자주 묻는 질문 (FAQ)

훈련 기록을 올렸는데 랭킹에 반영이 안 돼요.

훈련 기록은 승인관리자의 검토·승인 후 랭킹에 반영됩니다. 훈련 기록 → 📋 목록 탭에서 현재 승인 상태를 확인해 주세요.

클럽 가입 신청 후 아무 소식이 없어요.

클럽장이 아직 검토 중입니다. 클럽 상세 페이지에서 ⏳ 클럽장 승인 대기 중 상태가 표시되면 정상입니다. 오랜 시간이 지나도 처리가 없으면 마이페이지 → 관리자 문의를 이용해 문의하세요.

비밀번호를 잊어버렸어요.

로그인 화면 하단의 비밀번호를 잊으셨나요? 링크를 탭하세요. 이메일을 입력하면 재설정 코드가 자동으로 발송됩니다. 이메일이 계정에 등록되어 있지 않다면 마이페이지 → 관리자 문의 또는 관리자에게 직접 연락해 코드 발급을 요청하세요.

훈련 기록을 삭제하고 싶어요.

두 가지 방법으로 삭제할 수 있습니다. 피드에서 내 기록 카드 하단의 삭제 버튼을 누르거나, 훈련 기록 → 📋 목록 탭에서 기록 카드 우측 🗑️ 버튼을 누르세요.

기록에 올린 사진을 삭제하고 싶어요.

현재 사진 개별 삭제는 지원하지 않습니다. 기록 전체를 삭제한 뒤 사진 없이 다시 등록하는 방법을 이용하세요.

팔로우를 하면 상대방에게 알림이 가나요?

현재 푸시 알림 기능은 제공되지 않습니다.

클럽을 직접 만들 수 있나요?

관리자에게 클럽장 신청을 하여 승인을 받으면 클럽 화면에서 직접 만들 수 있습니다. 신청은 마이페이지 → 관리자 문의를 이용해 주세요.

여러 클럽에 동시에 가입할 수 있나요?

현재 한 개의 클럽에만 소속될 수 있습니다.

관리자에게 문의를 보냈는데 답장 확인은 어떻게 하나요?

마이페이지 → 관리자 문의 섹션에서 이전에 보낸 문의를 탭하면 관리자의 답장을 스레드 형태로 확인할 수 있습니다. 새 답장이 있으면 문의 항목에 답장 배지가 표시됩니다.
